How we can import a raster dataset using R

Patrick Laube , Nils Ratnaweera , Nikolaos Bakogiannis
12-03-2021

One of the important aspects of the upcoming exercise (Multi-Criteria Evaluation (MCE)) is the use and manipulation of raster datasets. In R, two are the main packages used to handle raster data: terra and raster package. The latter is still heavily used, but inevitably is going to be replaced by the first one.
Below we will use both of them to demonstrate how we can import a raster dataset. In the link below you can download a tif file, representing the Digital Elevation Model (Digitales Höhenmodell, DHM) of Canton Schwyz in Switzerland.

  • dhm25m.tif

Importing raster data with raster

library(raster)

raster_schwyz <- raster("dhm25m.tif") 
raster_schwyz # retrieving raster info

class      : RasterLayer 
dimensions : 1513, 1888, 2856544  (nrow, ncol, ncell)
resolution : 25, 25  (x, y)
extent     : 671937.5, 719137.5, 193512.5, 231337.5  (xmin, xmax, ymin, ymax)
crs        : +proj=somerc +lat_0=46.9524055555556 +lon_0=7.43958333333333 +k_0=1 +x_0=600000 +y_0=200000 +ellps=bessel +units=m +no_defs 
source     : dhm25m.tif 
names      : dhm25m

Importing raster data with terra

library(terra)

terra_schwyz <- terra::rast("dhm25m.tif")
terra_schwyz # retrieving raster info

class       : SpatRaster 
dimensions  : 1513, 1888, 1  (nrow, ncol, nlyr)
resolution  : 25, 25  (x, y)
extent      : 671937.5, 719137.5, 193512.5, 231337.5  (xmin, xmax, ymin, ymax)
coord. ref. : CH1903 / LV03 
source      : dhm25m.tif 
name        : dhm25m

Plotting raster data

To plot a raster dataset, we can simply use either of the following plot() functions:

  • terra::plot()
  • raster::plot()
terra::plot(terra_schwyz)
